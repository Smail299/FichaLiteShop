# FichaLiteShop
Этот плагин добавляет на сервер кастомный магазин /shop как на HolyWorld Lite. В нем можно как купить предмет, так и сделать ставку на предмет. Имеется всего 2 вида ставок, за сапфиры и монеты.
Иметься:

• Два формата покупки предметов, аукцион и обычная покупка
• В конфиге можно включить/выключить ранний вайп, тоесть при нем время до обновления ассортимента станет меньше
• В конфиге можно поменять название валют
• Имеется кнопка перехода в магазин китов и бустеров
• Купленные предметы перемещаться в /shop chest
